# Black Ramen Studio

A multipotentiality creative studio website built with Next.js 15, React 19, and Tailwind CSS 4.

## About

Black Ramen Studio is a creative practice that refuses to stay in one lane. We design, code, strategize, and tell stories across any medium that serves the idea.

## Tech Stack

- **Framework**: Next.js 15.5.3 with App Router
- **React**: 19.1.0
- **Styling**: Tailwind CSS 4
- **TypeScript**: Latest
- **Fonts**: Inter (sans-serif) + Libre Caslon Text (display)
- **Animations**: Framer Motion

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Build for Production

```bash
npm run build
npm start
```

## Features

- Dark theme optimized for creativity
- Animated photo slideshows
- Story deck with modal viewer
- Responsive design
- SEO optimized
- Fast performance with Turbopack

## Design Philosophy

The design embraces a dark, moody aesthetic inspired by black ramen — rich, complex, and full of depth. The color palette features warm accents (#ff6b35, #f77f00, #d62828) against a dark background (#0a0a0a).
